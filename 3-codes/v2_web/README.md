# 五子棋 Web 智能对局平台 (v2)

这是五子棋对局平台的 **v2** 版本，实现了基于 **暗黑磨砂玻璃质感 (Dark Glassmorphism)** 的 Web 界面，并将 v1 的 **Renju (禁手) 规则** 与 **对局保存** 功能无缝对接到网页中。

## 功能特性
1. **轻量级 API 架构**：
   - 使用 Flask 后端服务直接复用 Python `board.py` 算法，前端完全解耦只做渲染。
2. **磨砂玻璃视觉系统 (Glassmorphism)**：
   - 现代 Outfit 字体，炫酷深空灰与霓虹蓝/粉高光背景。
   - 珍珠与白玉材质感棋子，带微妙的 Radial Gradient 3D 渐变效果。
3. **Hover 智能落子辅助 (Hover Assist)**：
   - 鼠标在棋盘移动时提供半透明虚影定位。
   - 若当前格子触发黑棋的 **三三禁手、四四禁手或长连禁手**，虚影将自动变为**红色虚线**，并在上方浮现醒目的黄色警报框（例如：`⚠️ 黑棋: 三三禁手`），防止黑棋失误。
4. **完整的对局控制**：
   - 支持双人落子、随时悔棋 (Undo)、重新开始、导出 JSON 棋谱（符合 `save_flie.template.json`）。

---

## 快速启动

1. 安装依赖：
   ```bash
   pip install flask
   ```
2. 启动开发服务器：
   ```bash
   python app.py
   ```
3. 在浏览器中访问：`http://127.0.0.1:8000`
