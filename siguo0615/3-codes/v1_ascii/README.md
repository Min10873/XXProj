# 五子棋 ASCII 字符对战版 (v1)

这是五子棋对战平台的 **v1** 版本，实现了最核心的棋盘与胜负判定逻辑，支持本地双人对弈。

## 如何运行

确保你的系统中已安装 **Python 3**，然后在当前目录下运行：

```bash
python main.py
```

## 功能特点
1. **人性化的坐标支持**：
   - 字母加数字：如 `H8`, `h8`, `A1`, `O15` 等（字母大小写均可，表示第几列，数字表示第几行）。
   - 双数字输入：如 `8 8`, `1 15` 等（以空格或逗号分隔）。
2. **落子提示与高亮**：
   - 使用 `●` 表示黑子，`○` 表示白子，`·` 表示空位。
   - 上一步落子会高亮显示为 `●*` 或 `○*`，并伴有提示信息，方便玩家看清对手下在哪里。
3. **健壮性校验**：
   - 重复落子校验。
   - 越界输入及非合法坐标校验。
   - 支持通过输入 `exit` 或 `quit` 优雅退局。
