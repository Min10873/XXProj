## 4. 运行过程记录与问题解决
* **现象**：在根目录下直接运行 `python main.py` 报错 `C:\Python313\python.exe: can't open file 'C:\\AG\\siguo\\siguo0615\\main.py': [Errno 2] No such file or directory`。
* **原因**：当前工作目录（CWD）为项目根目录，而 `main.py` 实际上位于 `3-codes/v1_ascii/` 目录下。
* **解决方法**：
  * 指引用户使用根目录相对路径：`python 3-codes/v1_ascii/main.py`；
  * 或切换到对应目录后再运行：`cd 3-codes/v1_ascii` 紧接着 `python main.py`。
