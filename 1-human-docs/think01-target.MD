## 课题

五子棋对战平台

## 策略: 敏捷迭代+AI导师制

### 敏捷迭代

这条线是项目版本演进：

v1: ascii版
先把游戏基本规则跑通，不急着上 AI。

v2：加上web界面

v3：接入 LLM API
实现简单 AI 对话，形成人 vs AI的体验

v4：拟人的AI
通过记录不同人的风格，进行拟人化AI角色

### AI导师制

这条线是学习和工作方法：

人一开始的想法“不成熟、不清晰”，通过不断问 AI，把模糊想法变成阶段目标，再沉淀到飞书图里或 Markdown 文档里。

然后：

完成阶段目标，就进入下一步；
失败，就复盘、下调目标；
再推动、再迭代。

这条线解决的是：学生如何借助 AI，把自己从不会做项目，带到能持续推进项目。
